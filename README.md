## ![Target](https://img.shields.io/badge/-Core_Idea-E91E63?style=flat-square) Core Idea & Problem Statement

### The Problem We're Solving

In Bangladesh, over **160 million people** face a critical healthcare challenge: **branded medicines cost 2-5x more than generic alternatives** containing the exact same active ingredients.

**Real Example:**
| Medicine | Brand Name | Price | Generic Alternative | Price | Savings |
|----------|------------|-------|---------------------|-------|---------|
| Paracetamol 500mg | Napa | ৳2.00/tablet | Neopara | ৳0.50/tablet | **75%** |
| Omeprazole 20mg | Seclo | ৳8.00/capsule | Omag-DR | ৳2.50/capsule | **69%** |
| Amlodipine 5mg | Amlovas | ৳6.00/tablet | Amdocal | ৳1.50/tablet | **75%** |

**Why does this happen?**
- Doctors often prescribe expensive branded medicines
- Patients don't know cheaper alternatives exist
- Generic medicines are equally effective but less marketed
- No easy way to compare prices across 21,000+ medicines

### Our Solution

**Medicine Saver BD** is a mobile app that instantly shows all affordable generic alternatives for any medicine. Simply search for what your doctor prescribed, and see every equivalent option sorted by price.

### ![Search](https://img.shields.io/badge/1-Smart_Search-2196F3?style=flat-square&logo=searchengin) Smart Medicine Search

The core feature that makes finding alternatives effortless.

| Capability | Description | Technical Implementation |
|------------|-------------|-------------------------|
| **Instant Search** | Type any medicine name, see results in <100ms | SQLite FTS5 full-text search with indexing |
| **Fuzzy Matching** | Handles typos like "npa" → "Napa" | Levenshtein distance algorithm |
| **Bengali Support** | Search "প্যারাসিটামল" or "paracetamol" | Unicode normalization + bilingual database |
| **Voice Input** | Speak medicine names hands-free | Native Android SpeechRecognizer intent |
| **Strength Filters** | Filter by 500mg, 100ml, 10mg, etc. | Dynamic chip-based filtering |
| **Category Filters** | Filter by Tablet, Syrup, Injection, etc. | Dosage form categorization |
| **Sort Options** | Sort by price ↑↓, name A-Z, manufacturer | Multi-column sorting |
| **"You Searched For"** | Shows exact match price before alternatives | Exact name matching algorithm |
| **Recent Searches** | Quick access to last 5 searches | SharedPreferences persistent storage |

### Backend: Supabase (PostgreSQL + Auth + Storage)

| Service | Purpose | Details |
|---------|---------|---------|
| ![Auth](https://img.shields.io/badge/-Auth-blue?style=flat-square) | Google OAuth, email/password | JWT tokens, refresh tokens |
| ![DB](https://img.shields.io/badge/-Database-green?style=flat-square) | PostgreSQL for user data | `price_reports`, `user_cabinets` tables |
| ![Storage](https://img.shields.io/badge/-Storage-orange?style=flat-square) | Medicine database updates | `medicine_db` bucket for .db files |
| ![RLS](https://img.shields.io/badge/-RLS-red?style=flat-square) | Data protection | Users can only access their own data |

**Database Schema:**
```sql
-- Price Reports Table
CREATE TABLE price_reports (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  user_id UUID REFERENCES auth.users(id) NOT NULL,
  medicine_name TEXT NOT NULL,
  medicine_id INTEGER,
  price_paid DECIMAL(10,2) NOT NULL,
  mrp DECIMAL(10,2),
  pharmacy_name TEXT,
  location_area TEXT,
  upvotes INTEGER DEFAULT 0,
  is_verified BOOLEAN DEFAULT false,
  created_at TIMESTAMPTZ DEFAULT NOW()
);
```

## ![Cloud](https://img.shields.io/badge/-Updates-00BCD4?style=flat-square&logo=icloud) How Online Data Updates Work

### Database Update Flow

```
App Launch → Check local DB version → Query Supabase for latest version
    ↓
If newer version exists → Show update dialog → Download new DB
    ↓
Replace local medicines.db → Update version number → Use new data
```

### Update Frequency

| Update Type | Frequency |
|-------------|-----------|
| **Major Updates** | Monthly (new medicines, price changes) |
| **Minor Fixes** | As needed (corrections, additions) |
| **User Preference** | Manual check in Settings |
